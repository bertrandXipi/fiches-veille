# Prompt

## Définition

Dans le contexte de la veille technologique, un prompt est une instruction ou une commande textuelle structurée, fournie à un modèle d'intelligence artificielle générative, telle qu'un modèle de langage ou de génération d'images. Il vise à guider le comportement du modèle pour produire un résultat spécifique, que ce soit pour créer du contenu, analyser des données ou modifier une sortie existante. Son efficacité est cruciale pour explorer et exploiter les capacités des IA.
