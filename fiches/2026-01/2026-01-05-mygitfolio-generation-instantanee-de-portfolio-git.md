## Résumé

MyGitFolio est un outil permettant de générer instantanément une page de portfolio propre à partir d’un nom d’utilisateur GitHub. L’application affiche les dépôts, les statistiques et les points forts sans configuration préalable ni encombrement visuel.

Le projet a été développé en utilisant VS Code ainsi que les outils d’assistance au codage Claude et Codex. Il vise à simplifier la présentation des projets pour les développeurs souhaitant une vitrine rapide et efficace.

## Points clés

- Génération instantanée à partir du pseudonyme GitHub
- Aucune configuration requise (zéro setup)
- Développement assisté par IA (Claude/Codex)
